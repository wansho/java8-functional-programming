## 建模
```
- Artist 创作音乐的个人或团队。
	- name： 艺术家的名字（ 例如“ 甲壳虫乐队”）。
	- members： 乐队成员（ 例如“ 约翰 · 列侬”）， 该字段可为空。
	- origin： 乐队来自哪里（ 例如“ 利物浦”）。
	
- Album 专辑， 由若干曲目组成。
    - name： 专辑名（ 例如《 左轮手枪》）。
    - tracks： 专辑上所有曲目的列表。
    - musicians： 参与创作本专辑的艺术家列表。
    	
- Track 专辑中的一支曲目。
    - name： 曲目名称（ 例如《 黄色潜水艇》）。

```
